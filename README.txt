Sistema de Alquiler de Vehículos

Aplicación desarrollada en Java usando JOptionPane para la gestión de empleados, usuarios y vehículos dentro de una empresa de alquiler.

📌 Descripción

Este sistema permite administrar procesos básicos de alquiler mediante control de acceso por roles, registro de usuarios y visualización de vehículos.

Está construido con Programación Orientada a Objetos (POO), aplicando herencia y organización por clases.

⚙️ Funcionalidades

🔐 Inicio de sesión de empleados

👨‍💼 Control de roles (Administrador, Ingeniero, Asesor)

👥 Registro de usuarios

🚗 Visualización de vehículos

📂 Lectura de datos desde archivos .txt

🧠 Estructura del sistema

Usuario (hereda de Persona)
